## 四角色系統
- **車隊管理**：待處理決策中心（依緊急度排序）、AI 可執行建議、人力招募、區域比較、趨勢與原因、車隊排行
- **車隊負責人**：只管自己這一區的駕駛與待辦（紅黃綠 / 已通知次數 / 通知·語音·派交接）
- **司機**：我的安全分、今日任務、個人化提醒
- **貨主**：登入後只看自己的單，Uber 式即時追蹤（位置 + ETA）

> 行動皆為原型模擬，資料為 demo 假資料。
